3-Pressman, página 20, apresenta a relação, abaixo apesentada, de tipos/características de software. Explane cada um (Acrescente outros se desejar.)

Software Básico: coleção de programas escritos para dar apoio a outros programas.
Software de Tempo Real: software que monitora, analisa e controla eventos do mundo real.
Software Comercial: sistemas de operações comerciais e tomadas de decisões administrativas.            
Software Científico ou de Engenharia: caracterizado por algoritmos de processamento de números.
Software Embutido: usado para controlar produtos e sistemas para os mercados industriais e de consumo.
Software de Computador Pessoal: envolve processamento de textos, planilhas eletrônicas, diversões, etc.
Software de Inteligência Artificial: faz uso de algoritmos não numéricos para resolver problemas que não sejam favoráveis à computação ou à análise direta.
